#  Car Sales Data Analysis

##  Overview

This project performs **Exploratory Data Analysis (EDA)** on a car sales dataset to uncover patterns in pricing, brand popularity, mileage impact, and transmission trends.

---

##  Objective

* Analyze car sales data using Python
* Identify factors affecting car prices
* Understand brand distribution
* Visualize key insights

---

##  Dataset

The dataset includes:

* Year, Make, Model
* Body Type & Transmission
* Condition & Odometer
* Color & Interior
* Seller Information
* Market Price (MMR)

---

##  Tech Stack

* Python
* Pandas
* Matplotlib

---

##  Analysis Performed

* Top 10 car brands by sales
* Price distribution
* Odometer vs Price relationship
* Transmission type comparison

---

##  Key Insights

*  Hyundai and Infiniti have the highest sales count
*  Most cars fall in the 10K–30K price range
*  Higher mileage reduces car price
*  Automatic cars dominate the dataset

---

##  Conclusion

* Mileage is a major factor influencing price
* The dataset is dominated by mid-range cars
* Automatic transmission is more common
* Certain brands show higher market activity
